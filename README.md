# encuestas - Encuestas en Línea con Struts2

Proyecto creado a partir del arquetipo Maven `struts2-archetype-blank` para desarrollar una aplicación web de encuestas en línea utilizando el framework Struts2.

```shell
mvn archetype:generate -B -DgroupId=gal.jairochapela -DartifactId=encuestas 
-DarchetypeGroupId=org.apache.struts -DarchetypeArtifactId=struts2-archetype-blank
```
